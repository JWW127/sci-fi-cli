### Install

##### Option 1 Global

`npm install -g sci-fi-cli`

##### Option 2 Clone Repo & Install

1.  `git clone https://github.com/JWW127/sci-fi-cli.git`
2.  `cd sci-fi-cli && npm install`

### Last call and use options

- get help menu
  `sci --help`
- get one sci-fi quote
  `sci --one`
- get many sci-fi quotes
  `sci --many`
